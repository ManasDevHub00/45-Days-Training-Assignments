### 10. Frequency Counter Function

Function:

```python
count_frequency(arr)
```

Example:

```python
arr = [1, 2, 2, 3, 1, 4, 2]
```

Output:

```text
1 -> 2 times
2 -> 3 times
3 -> 1 time
4 -> 1 time
```

---

### 11. Duplicate Elements Finder

Function:

```python
find_duplicates(arr)
```

Example:

```python
arr = [10, 20, 30, 20, 40, 10, 50, 30]
```

Output:

```text
Duplicate elements are:
10
20
30
```
